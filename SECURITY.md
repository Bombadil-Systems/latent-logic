# Security Policy

## Reporting Vulnerabilities

If you discover a security issue in Latent Logic itself, email chris@bombadil.systems. I read everything and will respond within 48 hours.

## Intended Use

This tool is intended for authorized penetration testing, red team engagements, and security research. Do not use it against systems you don't own or have written permission to test.

## Responsible Disclosure

If you use Latent Logic and discover a gap in a commercial security product, please follow responsible disclosure with the affected vendor.
